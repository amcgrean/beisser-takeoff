## Critical TS pattern тАФ ALWAYS use this for postgres.js results

```typescript
// CORRECT тАФ db.execute() and getErpSql() return RowList, not { rows: [...] }
const rows = await sql`SELECT ...` as unknown as MyType[];
// or
const rows = (await db.execute(sql`SELECT ...`)) as unknown as MyType[];

// WRONG тАФ will fail build with "Property 'rows' does not exist on type 'RowList'"
const { rows } = await sql`SELECT ...`;
```
